# SloughGPT Examples

Example scripts demonstrating SloughGPT features.

## Running Examples

From the **repository root**, after `pip install -e ".[dev]"` (or `pip install -e .`):

```bash
cd scripts/examples

python3 train_example.py
python3 inference_example.py
python3 benchmark_example.py
```

## Examples

### train_example.py
Demonstrates how to train a custom NanoGPT model with the optimized trainer.

### inference_example.py
Shows how to use the inference engine for text generation.

### benchmark_example.py
Runs a simple benchmark to test GPU performance.
