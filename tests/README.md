## Tests

Pytest collection lives under this directory (see **`pytest.ini`** at the repo root).

- **Top-level** — `test_*.py` modules by area (API, SDK, training, inference, security, etc.).
- **`server/`** — additional HTTP/API coverage (`test_server_api.py`).
- **Markers** — `unit`, `integration`, `e2e`, `slow` (see `pytest.ini`).

From the repo root: `python3 -m pytest tests/` (full suite) or the subset used in **`.github/workflows/reusable-ci-core.yml`** (includes **`test_checkpoint_utils`**, **`test_config`** (**`merge_args_with_config`**, **`get_device`**), **`test_wandb_helpers`**, **`test_trainer_protocol`** (**`TrainerProtocol`**, **`run_trainer_async`**), **`test_train_sloughgpt_generate_text`**, **`test_train_sloughgpt_resume`**, **`test_sloughgpt_trainer_smoke`**, **`test_sloughgpt_trainer_resume`**, **`test_sloughgpt_trainer_progress_callback`**, **`test_cli_train_export_stem`**, **`test_cli_train_api_payload`**, **`test_training_router_kwds`**, **`test_training_schemas`**, **`test_lm_eval_char`**, **`test_cli_local_soul_candidates`**, **`test_soul_engine_conversation`**, **`test_repo_root_package_json`** (root **`package.json`**, **`dev:stack`**, **`verify.sh`**, **`make test-repo-root`**; optional **`npm run test:repo-root`**), **`test_sloughgpt_colab_notebook`** — Colab **`sloughgpt_colab.ipynb`** hooks, **`pyproject.toml`** **`notebook`** optional extra, smoke script contents, **`.gitignore`** nested **`sloughGPT/`** rule, and **`bash …/run_colab_notebook_smoke.sh --help`** (**skipped** if **`bash`** is not on **`PATH`** — e.g. native Windows; use Git Bash/WSL locally if you want that case to run), plus **`python3 train_sloughgpt.py --help`**). Optional: `./verify.sh` for path checks, ruff smoke when **`ruff`** is installed, and **`apps/web`** **`npm run ci`** when **`node_modules`** exists. **`python3 -m pip install -e ".[notebook]"`** pulls Jupyter/nbconvert for **`scripts/run_colab_notebook_smoke.sh`** / **`make colab-smoke`** (see **README.md** *Google Colab*). **`make colab-test`** runs only **`tests/test_sloughgpt_colab_notebook.py`**.

**Also in `ci_cd.yml` (not the `reusable-ci-core` pytest subset):** **`test-web`** (**`npm run ci`** in **`apps/web`**), **`test-strui`** (**`npm run ci`** in **`packages/strui`**), and **`test-sdk-ts`** (**`npm run ci`** in **`packages/sdk-ts/typescript-sdk`**). **`sdk-test-py`** runs **`tests/test_sdk.py`** only. **`standards-schemas`** validates example manifests (**`scripts/validate_standards_schemas.py`**). See **CONTRIBUTING.md** for local parity commands.
