# Contributing

Thanks for helping improve SloughGPT.

## Quick start

1. **Branch** from `main` with a descriptive name.
2. **Install** from the repo root: `pip install -e ".[dev]"` (see **QUICKSTART.md**).
3. **Validate**: `./verify.sh` and `python3 -m pytest tests/` (CI runs a subset in **`.github/workflows/reusable-ci-core.yml`**).
4. **Open a PR** with a clear description of intent and scope.

See **docs/STRUCTURE.md** and **docs/MIGRATION.md** for layout and migration history.
